## 📌 Project Overview

Binary image classification system that detects **potholes** in road images using three deep learning architectures trained from scratch and via transfer learning. The system includes a live Gradio web interface with **Grad-CAM explainability**.

> 🔗 **Live Demo:** [Hugging Face Spaces – add your link here]

## 🗂️ Repository Structure

```
pothole-detection/
├── app.py                # Gradio web interface (HuggingFace Spaces)
├── train.py              # Training pipeline (all 3 models)
├── evaluate.py           # Test-set evaluation + GradCAM
├── predict.py            # PotholeDetector class (single image inference)
├── models.py             # All 3 model architectures
├── dataset.py            # Data loading & augmentation
├── config.py             # All hyperparameters & paths
├── utils.py              # Shared helper functions
├── requirements.txt      # Python dependencies
├── saved_models/         # Model checkpoints (.pth) — generated by train.py
├── results/              # Plots, confusion matrices — generated by evaluate.py
└── examples/             # Sample images for the Gradio demo
```

## 🧠 Models

| Model | Description | Parameters (approx.) |
|---|---|---|
| **Custom CNN** | 5-block CNN built from scratch, BatchNorm + Dropout | ~4.5M |
| **VGG16** | Pre-trained ImageNet, fine-tuned block-5 + custom head | ~138M |
| **MobileNetV2** | Lightweight, inverted residuals, edge-deployment ready | ~3.4M |

## ⚡ Quickstart

### 1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/pothole-detection.git
cd pothole-detection
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Place your dataset

Copy your `Dataset/Dataset/` folder into the project root so the structure matches the layout shown above.

### 4. Train all models

```bash
python train.py
```

This trains all three models sequentially and saves the best checkpoints to `saved_models/`.  
Training curves are saved to `results/`.

### 5. Evaluate on the test set

```bash
python evaluate.py
```

Generates confusion matrices, ROC curves, Grad-CAM visualisations, and a metrics comparison chart — all saved to `results/`.

### 6. Run the Gradio demo locally

```bash
python app.py
```

Opens a browser window at `http://127.0.0.1:7860`.  
Add `share=True` in `app.py` → `demo.launch()` for a public link.

### 7. Single image prediction

```bash
python predict.py path/to/road_image.jpg mobilenetv2
```

   - `saved_models/` folder (contains the `.pth` checkpoint files)

5. Copy the public URL and paste it into your report.

## 📊 Training Configuration

| Hyperparameter | Value |
|---|---|

| Learning rate | 1e-4 |

## 🔬 Key Features

- **3 architectures** benchmarked under identical conditions
- **Aggressive augmentation** — flip, rotate, colour jitter, shear, random erasing
- **Grad-CAM explainability** — visual heatmaps showing model attention
- **ROC curves + AUC** for all models
- **Early stopping** prevents overfitting on the small dataset
- **PotholeDetector class** — clean API for production inference
- **Gradio app** — browser-based demo with confidence chart
